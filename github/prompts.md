# GitHub - GenAI Prompts

> This file contains suggested prompts for use with GenAI tools like ChatGPT, Gemini, Claude, etc. Use these as starting points and customize them for your specific needs.

## Use Case 1: Git Fundamentals and Workflow

### Prompt
```
I'm new to Git and GitHub. Can you explain:
1. What's the difference between Git and GitHub?
2. What are the basic Git commands I need to know (clone, add, commit, push, pull)?
3. What is a branch and why should I use it instead of committing to main?
Include practical examples for a typical development workflow.
```

### Expected Output
- Clear distinction between Git and GitHub
- Essential Git commands with explanations
- Branching strategy explanation

---

## Tips for Better Results
- Be specific about your Git workflow and team setup
- Include your GitHub organization structure if relevant
- Ask for command-line examples if you prefer terminal usage
- Request GitHub UI instructions if you prefer the web interface
- Include security concerns when asking about access control
- Ask about specific error messages you encounter
